# Deep Learning Projects: Leveraging `fastai v2`

This repository contains different deep learning project notebooks that were run using the fastai framework. 

## Structure

Here is a list of the projects contained in this repository, classified according to the domain they belong to.

**Vision:**

- **[rainforest-audio](https://github.com/luca-martial/fastai-v2-projects/tree/master/mask-wear)**: The goal of this project was to build a machine learning model that predicts which species are present in rainforest audio recordings. My best submission scored a LWLRAP of 0.7
